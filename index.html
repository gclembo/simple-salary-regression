Simple Salary Regression in R

For this project I am utilizing my R programming skills to fit a linear regression model to data. This is a simple linear regression prediction model using data from the "Salary Dataset - Simple linear regression" Kaggle dataset. This dataset from user Allena Venkata Sai Abhishe and can be found here. This provides data describing years of experience and the salary of employees and is meant to be used to the basics of linear regression. As I have seen, the data have a strong linear association. This makes a linear regression model apt for predicting employee salary based on years of experience. Click below to see the code markdown walking through my processes.
